    インストール

    brew install heroku-toolbelt

    ログイン

    heroku login

    概要

    • アプリケーションを稼働させるためのプラットフォーム
    • PaaS (Platform as a Service)
    • heroku.com

    必要知識

    • Ruby on Rails / Ruby
    • Unix コマンド
    • git
    • posgreSQL

    環境

    • ローカル開発環境を作る

    PaaSとは何か？

    • 今まではローカル開発環境内で作り，サーバに送る
    • サーバ内も同じ環境を作る必要がある，また，セキュリティや分散用のシステムも必要だった
    • PaaSでは，ソースコードにいくつかの設定ファイルをつけるだけでHerokuで環境が自動で作られるようになった．
    • Dependencies : 依存関係
    • Procfile : 実際にheroku上で動かしたいコマンド一覧
    • gitのコマンド一発でherokuに送ることができる
    • Slug : アプリケーションをさっと稼働することができる
    • Dyno : 一時的なデータ
    • Addon : メールや分析ツールアドオン
    • JSやCSSや画像は別サービスを利用する方が良い
