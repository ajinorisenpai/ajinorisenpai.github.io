    Anaconda

    仮想環境を作る

    conda create -n py27 python=2.7 anaconda

    仮想環境の確認

    conda info -e

    仮想環境の削除

    conda remove -n py27 -all

    仮想環境をActiveにする

    source activate py27 自前の環境だと activate gpy27
